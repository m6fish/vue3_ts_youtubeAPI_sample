# vue3_ts_youtubeAPI_sample

Use Vue3 + typescript + composition api + Youtube video-API to make a basic RWD project.

---

### How

>$ npm install

>$ npm run serve

### Set youtube API-key

provide an .env file

> VUE_APP_VIDEO_KEY={$your_key}
